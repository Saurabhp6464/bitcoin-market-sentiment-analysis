
# Bitcoin Market Sentiment Analysis

## Project Overview

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using historical trading data from Hyperliquid.

## Objective

- Analyze trader performance under different market sentiments.
- Compare profit and loss during Fear and Greed periods.
- Identify profitable trading patterns.
- Generate business insights through Exploratory Data Analysis (EDA).

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Workflow

1. Import Libraries
2. Load Datasets
3. Data Cleaning
4. Data Preprocessing
5. Merge Datasets
6. Exploratory Data Analysis
7. Visualization
8. Key Findings
9. Conclusion

## Output

The notebook provides visualizations and insights into how market sentiment may influence trader performance.
